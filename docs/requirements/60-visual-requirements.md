# ビジュアル設計 — 見下ろしマップの見栄え向上ガイド

## P（結論）

ドラクエ風の見下ろしマップは、
**「視覚的な気持ちよさ」＋「迷わない構造」＋「世界に入り込める演出」**の3つを強化する手法を組み合わせることで、見栄えと体験が大きく向上します。

---

## R（理由）

見下ろしマップは情報量が限られるため、

- 情報が少ない → 退屈になりやすい
- 情報が多すぎる → 分かりにくい

というトレードオフがあります。

そこで「見せ方の工夫」を使うことで、
脳の負担を減らしつつ、ワクワク感を増やすことが重要になります。

---

## E（具体例：手法一覧＋効果）

---

### ① タイルのバリエーション（同じ地形に変化をつける）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 視覚に敏感な人（「単調だな」と感じやすい人） |
| **作用** | 「同じ草でも違う」と感じさせる（錯覚） |
| **和らげる問題** | 単調さ・飽き |
| **メリット** | 少ない素材でも豊かな世界に見える / 開発コストが低い |

---

### ② 高低差・段差の表現（立体感）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 空間をイメージしたい人 |
| **作用** | 「奥行き」を感じる |
| **和らげる問題** | 平面的でつまらない感じ |
| **メリット** | 探索したくなる / ルート設計がしやすい（自然な道誘導） |

---

### ③ ランドマーク配置（目印）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 迷いやすい人 |
| **作用** | 「ここを目指せばいい」と分かる |
| **和らげる問題** | 迷子・方向感覚の喪失 |
| **メリット** | ストレス減少 / 記憶に残るマップになる |

---

### ④ 色のグラデーション・ライティング

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 感情でゲームを楽しむ人 |
| **作用** | 雰囲気（安心・不安・神秘）を作る |
| **和らげる問題** | 無機質さ・没入感の低さ |
| **メリット** | 世界観が一気に深くなる / ストーリーと連動できる |

---

### ⑤ 動く要素（アニメーション）

| 項目 | 内容 |
|------|------|

| **作用** | 「生きている世界」と感じる |

| **メリット** | 視線誘導にも使える / 長時間プレイでも飽きにくい |

### ⑥ 道のデザイン（自然な誘導）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | ゲームに慣れていない人 |
| **作用** | 無意識に正しい方向に進ませる |
| **和らげる問題** | 次に何をすればいいか分からない |
| **メリット** | 説明なしで進行できる / UXが良くなる |

---

### ⑦ 密度のコントロール（情報の強弱）

| **作用** | 「重要な場所」が目立つ |

| **メリット** | 視線誘導 / プレイヤーの集中力維持 |

---

### ⑧ インタラクションの予感（触れそう感）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 探索好きな人 |
| **作用** | 「これ調べられそう」と思わせる |

| **メリット** | プレイヤーが自発的に動く / 没入感UP |

---

## P（まとめ）

見下ろしマップの本質は、

| レイヤー | 手法 |
|----------|------|
| **見た目を良くする** | タイル・色・アニメ |
| **迷わせない** | 道・ランドマーク |
| **世界に入り込ませる** | 動き・密度・触れそう感 |

の3レイヤー設計です。

つまり、
**「美しさ」ではなく「脳の負担を減らしながらワクワクさせる設計」**が本質です。
