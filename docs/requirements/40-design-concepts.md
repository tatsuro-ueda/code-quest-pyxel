# デザインコンセプト集

## シズル感のあるピクセルアートの5ルール

### P（結論）

シズル感のあるピクセルアートは、
「光・影・色・動き」を“ルール化”して入れるだけで再現できる。
つまり、センスではなく手順で作れる。

### R（理由）

シズル感の正体：
- 光がある（ハイライト）
- 影がある（立体感）
- 色に差がある（コントラスト）
- 少し動く（生命感）

これを入れると、脳が勝手に「質感」を感じる。

### E（具体：5つのルール）

#### ① ハイライトを必ず入れる（最重要）
- ルール: 左上から光が当たると決める。明るい色を1〜2ドット置く
- 効果: 一気に「ツヤ」が出る

#### ② 影を必ずセットで入れる
- ルール: ハイライトの反対側（右下）に暗い色を置く
- 効果: 平面→立体に変わる

#### ③ 3色ルール（少なすぎず多すぎず）
- ルール: 1つの物体につき 「明るい・中間・暗い」 の3色だけ使う
- 効果: ごちゃつかずリッチに見える

#### ④ 輪郭を少し崩す（ギザギザを利用）
- ルール: 完全な四角・直線を避ける。1ドットずらす
- 効果: 「作り物感」が消える、自然っぽくなる

#### ⑤ 1〜2フレームだけ動かす（超重要）
- ルール: 2コマだけ作る、ほんの少し変える（1〜2ドット）
- 効果: 一気に「生きてる感」が出る、シズル感の決定打

### 補助ルール
- ⑥ 周囲を少し暗くする → 主役が目立つ
- ⑦ キラッ（白1ドット点滅） → 宝箱・アイテムに使う
- ⑧ 色相を少しずらす → 影を「暗く」するだけでなく「少し青寄り」にする

### P（まとめ）

シズル感は、
1. 光（ハイライト）
2. 影（シェーディング）
3. 色（3色制限）
4. 形（少し崩す）
5. 動き（2フレーム）

この5つで作れる。
**「上手い人のセンス」ではなく「再現可能な設計」**である。
