# オーディオ設計 — 見下ろしRPGにおけるBGMの役割ガイド

## P（結論）

見下ろしRPGにおけるBGMの役割は、
**「感情を誘導する」「行動を支える」「世界観を記憶させる」**の3つです。

つまり、BGMは"雰囲気づくり"ではなく、プレイヤーの認知と行動をコントロールする装置です。

---

## R（理由）

人間は視覚だけでなく、音によっても判断しています。

- 明るい音 → 安心して探索する
- 不穏な音 → 警戒する
- テンポが速い → 急ぐ・戦う

このようにBGMは、
説明なしでプレイヤーの状態（気持ち・行動）を変えることができます。

見下ろしRPGは情報が少ない分、
音の影響力が特に大きいのが特徴です。

### ① エリアごとのテーマ曲（場所の記憶）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 世界観を楽しみたい人 |
| **作用** | 「この曲＝この場所」と結びつく |
| **和らげる問題** | マップが似ていて区別しにくい |
| **メリット** | 記憶に残る世界になる / 少ないグラでも差別化できる |

---

### ② テンポ・リズムの調整（行動速度の誘導）

| 項目 | 内容 |
|------|------|
| **作用するユーザー** | 無意識に行動するすべての人 |
| **作用** | ゆっくり → 探索・安心 / 速い → 緊張・戦闘 |
| **和らげる問題** | 行動の迷い（ゆっくり見るべきか？急ぐべきか？） |
| **メリット** | プレイヤーを自然に誘導できる / UI説明が減る |

| **和らげる問題** | 敵やイベントの前触れが分かりにくい |
| **メリット** | サプライズと納得感の両立 / ホラー・ボス前の演出強化 |

## P（まとめ）

BGMは単なる"音楽"ではなく、

- **感情を作る**（安心・緊張）
- **行動を誘導する**（探索・戦闘）
- **記憶を作る**（場所・イベント）

という役割を持ちます。

つまり、
**「説明を減らし、プレイヤーを自然に動かすための設計ツール」**です。
