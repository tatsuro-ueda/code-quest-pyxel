# language: ja

Feature: ブロッククエスト 〜コードの王国を救え〜 受け入れ条件
  Scratchテーマのドラゴンクエスト風RPGとして
  すべてのゲームシステムが正しく動作し
  PC（キーボード）とスマホ（仮想パッド）の両方でプレイできること

  # ============================================================
  # 1. タイトル画面
  # ============================================================

  Scenario: タイトル画面の表示
    Given ゲームをロードした
    Then タイトル「ブロッククエスト」が表示される
    And 「〜コードの王国を救え〜」が表示される
    And 「はじめから」「つづきから」のメニューが表示される

  Scenario: 「はじめから」を選択して新規ゲーム開始
    Given タイトル画面が表示されている
    When 「はじめから」にカーソルを合わせてAボタンを押す
    Then マップ画面に遷移する
    And プレイヤーのレベルが1である
    And プレイヤーの武器が「マウス」である
    And プレイヤーの防具が「きほんのちしき」である

  Scenario: 「つづきから」でセーブデータをロード
    Given localStorageにセーブデータが存在する
    And タイトル画面が表示されている
    When 「つづきから」にカーソルを合わせてAボタンを押す
    Then セーブ時のプレイヤー状態でマップ画面に遷移する

  Scenario: 「つづきから」でセーブデータが無い場合
    Given localStorageにセーブデータが存在しない
    And タイトル画面が表示されている
    When 「つづきから」にカーソルを合わせてAボタンを押す
    Then 「セーブデータがありません」というメッセージが表示される
