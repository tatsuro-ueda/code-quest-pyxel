# プレイスルーテキスト（最終版）

ゲーム開始からエンディングまでにプレイヤーが読む全テキストを、
プレイ順に並べたもの。story-design.md の変更をすべて反映済み。

5分クリア想定。全体で15〜20戦。経験値稼ぎ不要。
テキストは読ませる前提。短さより深さを優先。
