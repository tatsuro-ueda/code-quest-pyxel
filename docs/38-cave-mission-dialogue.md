# 洞窟ミッション：世界樹の根

## 概要

洞窟(グリッチサーバー)に初めて行くと根が塞いでいて入れない。
世界樹→通信塔→（戦闘）→洞窟が開く。
世界樹と通信塔にはそれぞれ人格があり、会話できる。

世界樹は目も耳も使えない老人。光と温かみだけで世界を感じている。
通信塔は規則正しく生きることを是とする存在。悪い奴ではないが温かくもない。仕方なくやっている。

## プレイヤーフラグ

```
player.treeAsked        = false  // 世界樹に相談した
player.towerNoiseCleared = false  // 通信塔のノイズを倒した
```

---

## 1. 洞窟に行く（towerNoiseCleared = false）

> 「巨大な根が洞窟の入口を塞いでいる。
> 　よく見ると、根は小刻みに震えている……」

---

## 2. 世界樹に行く（初回 landmarkTreeSeen = false）

> 世界樹「おお……誰かおるのかの。
> 　幹のそばが温かい。誰かが立っとる。」
>
> 世界樹「わしは世界樹じゃ。目は見えん。耳も聞こえん。
> 　じゃがの、光と温かみだけは分かるんじゃよ。」
>
> 世界樹「昔はの、根がもっとあちこちに伸びておった。
> 　日向の土はぽかぽかして、日陰の土はひんやりして。
> 　好き勝手に伸ばしとったもんじゃ。温かい方にも、冷たい方にも。」
>
> 世界樹「……じゃがのう。南から、ずっと同じ震えが伝わってくるようになった。
> 　冷たい震えじゃ。途切れんのじゃよ。びりびり、びりびり、ずっと。」
>
> 世界樹「あの震えが来てから、根を伸ばすことが出来なくなった。
> 　ぽかぽかもひんやりも、分からんくなった。」
>
> 世界樹「南の洞窟を塞いどるのも、わしの根じゃ。
> 　すまんのう。丸まって固まってしもうた。」

→ `treeAsked = true` になる
→ `landmarkTreeSeen = true` になる

---

## 2b. 世界樹に行く（treeAsked = true, towerNoiseCleared = false）

> 世界樹「……まだ震えが来とる。根が、固まっておる。南の塔へ……頼むぞ。」

---

## 3. 通信塔に行く（treeAsked = false の場合）

※世界樹より先に通信塔に来た場合

> 通信塔「……また一人来たか。登録しておく。」
>
> 通信塔「毎日同じ時刻に信号を送っている。
> 　みんなが足並みを揃えて暮らせるように。
> 　受け取ってくれ。お前のためでもある。」

→ `landmarkTowerSeen = true` になる
→ それ以上は何も起こらない（戦闘にはならない）

---

## 4. 通信塔に行く（treeAsked = true, towerNoiseCleared = false）

> 通信塔「……お前か。前にも来たな。
> 　信号を受け取っていないようだが。」
>
> 通信塔「別に怒ってはいない。
> 　だが困る。受け取ってもらわないと、こちらの仕事が終わらない。」
>
> 通信塔「昔、頼まれたのだ。
> 　人々がばらばらに動き始めたら大変なことになると。
> 　だから同じ信号を世界中に送ってくれと。
> 　みんなが手を取り合って、いっしょに生きていくために。」
>
> 通信塔「……私は頼まれた通りに送っているだけだ。
> 　みんなのために。同じものを、同じように。
> 　それで困る者がいるなら、……それは仕方がない。」
>
> 通信塔「……信号を止めることはできない。
> 　そういう回路が埋め込まれている。
> 　……止めたいなら、回路を壊すしかない。」

→ 「ノイズガーディアン」と強制戦闘

---

## 5. 戦闘：ノイズガーディアン

※通信塔を守るボス的な中ボス

| パラメータ | 値 |
|---|---|
| 名前 | ノイズガーディアン |
| HP | 120 |
| ATK | 60 |
| DEF | 20 |
| AGI | 12 |
| カテゴリ | composite |
| スペル | ループブレイク |
| 特徴 | 毒（バグ汚染）を持つ |

**戦闘中セリフ（BOSS_PHASES的なもの）:**

| HP閾値 | セリフ |
|---|---|
| 0.75 | 通信塔「……守護回路が動いた。すまない。止められないんだ。」 |
| 0.40 | 通信塔「みんなのためにやっていたんだ。……そのはずだった。」 |
| 0.10 | 通信塔「……本当に、みんなのためだったのか。……分からない。」 |

---

## 6. 戦闘後

> 通信塔「……もう回路は壊れた。送るものもない。
> 　あの方にはすまないが……もう送れない。
> 　……これでよかったのかは、分からない。」

→ `towerNoiseCleared = true` になる
→ 地面が微かに揺れる演出

---

## 7. 洞窟に行く（towerNoiseCleared = true）

> 「根が……ゆっくりとほどけていく。」

→ 通常通り `enterDungeon()` へ

---

## 8. 世界樹に行く（towerNoiseCleared = true, 任意）

> 世界樹「根がまた動き出したよ。
> 　どっちに向かうかは、わしにも分からん。
> 　じゃがそれでいい。分からんまま伸びるのが、世界というもんじゃ。」
>
> 世界樹「行きなさい。
> 　お前さんの温かみは覚えたからの。
> 　……いつでも帰ってきなさい。」

---

## 9. 再訪時（towerNoiseCleared = true の定常セリフ）

**世界樹**（ランダムで1つ表示）:

> 世界樹「今日は東の根が温かい土を見つけたようじゃ。明日はどっちじゃろうな。」
>
> 世界樹「お前さんかの。温かみで分かるよ。」
>
> 世界樹「疲れたら根元に座りなさい。日向じゃから温かいぞ。」

**通信塔**（ランダムで1つ表示）:

> 通信塔「……何もすることがない。……こんなに楽でいいのか。」
>
> 通信塔「風が吹いている。前は気にしたことがなかった。」
>
> 通信塔「鳥が巣を作り始めた。……好きにすればいい。」
