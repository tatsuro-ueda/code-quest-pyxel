# Code Maker bundle manifest (Q4B: concat 生成型)
#
# bundler は本ファイルを上から順に読み、各 .py の内容を連結して
# Code Maker 用 single-file main.py の CORE_BLOCK を構築する。
#
# ルール:
# - 依存関係の葉（定数）から順に並べる。先に定義された symbol は
#   後から出現しても参照できる
# - `from src.X import *` / `from src.X.Y import ...` 行は bundler が除去する
#   （連結後は同一スコープに全部入るため import 不要）
# - `from __future__ import annotations` / `import pyxel` / `import random`
#   などの標準系 import は bundler が dedup する
# - src/runtime/main_runtime.py は含めない（shim そのもの）
# - src/runtime/main_development_runtime.py は Phase 3 で削除予定・含めない
# - '#' 始まりと空行はコメント/区切りとして無視
